# YouTube RAG Chatbot

## Description
This project is a Retrieval-Augmented Generation (RAG) chatbot built using LangChain, FAISS, HuggingFace Embeddings, and Groq LLM.

## Features
- Extracts YouTube transcript
- Splits text into chunks
- Creates vector embeddings
- Stores embeddings in FAISS
- Retrieves relevant context
- Answers user questions using Groq LLM

## Tech Stack
- Python
- LangChain
- FAISS
- HuggingFace Embeddings
- Groq API
- YouTube Transcript API
